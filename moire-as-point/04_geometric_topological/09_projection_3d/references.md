# References

## Artists
- **Felice Varini** — Architectural anamorphic installations. The observer's position completes the geometry — similar to how projection moiré requires specific viewing angles.
- **Olafur Eliasson** — "The Weather Project" (2003). Light and shadow as sculptural material.
- **Anthony McCall** — "Line Describing a Cone" (1973). Projected light as solid form. The projection-as-object principle.

## Scientific
- **Shadow Moiré Interferometry** — Industrial measurement technique. Accuracy to microns using simple grid projection.
- **Projection Moiré** — Uses projected light patterns instead of physical grids. Digital light processing (DLP) enables dynamic pattern generation.
- **Phase-Shifting Profilometry** — The gold standard for 3D scanning. Multiple phase-shifted patterns eliminate ambiguity.
- **Structured Light Scanning** — Kinect, iPhone FaceID. All use moiré-like interference principles.

## Historical
- **Conte's Shadow Method** — 19th century topographic mapping using shadow lengths.
- **Moire Topography** — 1970s medical application: mapping body surface shape for orthopedics and plastic surgery.
- **Holographic Interferometry** — 1960s onward. Uses wave interference (not grid moiré) but same contour-line principle.
