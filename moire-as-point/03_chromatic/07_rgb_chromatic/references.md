# References

## Artists
- **Carlos Cruz-Diez** — "Physichromie" series (1959-present). Chromatic interference using slatted structures. The physical analogue of RGB moiré.
- **Julio Le Parc** — "Continuel-lumière" (1960s). Light installations with chromatic interference.
- **Olafur Eliasson** — "Your Colour Memory" (2004). Color as temporal/spatial experience.
- **James Turrell** — Light and color as primary media. Not moiré per se, but the same obsession with color as phenomenon.

## Scientific
- **Fourier Color Theory** — Color perception is fundamentally interferential. The eye's cone cells are spatial frequency detectors.
- **Opponent Process Theory** — Red-green and blue-yellow are opponent channels. RGB moiré directly maps to this neural architecture.
- **Bezold-Brücke Effect** — Hue shifts with intensity. High-contrast moiré fringes naturally exhibit this.
- **Helmholtz-Kohlrausch Effect** — Saturated colors appear brighter. Chromatic moiré can exploit this for perceived luminance variation without actual luminance change.

## Historical
- **Pointillism** — Seurat, Signac. Color mixing through spatial separation rather than blending. The analog ancestor of digital chromatic moiré.
- **Divisionism** — Neo-Impressionist technique. Scientific color theory applied to painting.
- **Halftone Printing** — 1880s onward. CMYK dots at different angles create moiré if misaligned — the "problem" that chromatic moiré weaponizes.
