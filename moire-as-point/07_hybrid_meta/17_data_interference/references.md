# References

## Artists
- **Jeremy White** — NYT data visualization. Uses moiré-like patterns for complex data relationships.
- **Moritz Stefaner** — "Truth & Beauty Operations." Data visualization as aesthetic practice.
- **Nicholas Felton** — "Annual Reports." Personal data as visual art — the data-as-self principle.
- **Giorgia Lupi & Stefanie Posavec** — "Dear Data." Hand-drawn data visualization — the analog ancestor of data moiré.

## Scientific
- **Correlation Analysis** — Pearson, Spearman, mutual information. The statistical foundation of what data moiré visualizes.
- **Principal Component Analysis** — PCA finds the "main axes" of data variation. Data moiré finds the "main beats" of data interference.
- **Fourier Analysis of Data** — Any dataset can be decomposed into frequency components. Data moiré is the spatial equivalent.
- **Information Theory** — Shannon entropy, mutual information. The moiré encodes information-theoretic relationships.

## Historical
- **John Snow's Cholera Map** — 1854. The foundational data visualization — mapping disease to water pumps.
- **Charles Minard** — Napoleon's Russian campaign map (1869). Multi-variable data visualization — the ancestor of multi-dataset moiré.
- **Jacques Bertin** — "Sémiologie Graphique" (1967). The grammar of graphics — the theoretical foundation of data visualization.
