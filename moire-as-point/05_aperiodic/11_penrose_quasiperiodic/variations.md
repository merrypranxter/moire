# Variations

## 1. Ammann-Beenker Tiling
Use the 8-fold Ammann-Beenker tiling instead of 5-fold Penrose. The moiré has octagonal symmetry — different "flavor" of quasiperiodicity.

## 2. Fibonacci Word Moiré
Use the Fibonacci word (a binary sequence generated by substitution) to modulate grid presence:
```glsl
float fibonacciWord(int n) {
    // L-system: 0 -> 01, 1 -> 0
    // Use n to index into the infinite word
}
```
The moiré follows the Fibonacci rhythm — long/short spacings in golden ratio.

## 3. Cut-and-Project Method
Generate quasiperiodic patterns by projecting a higher-dimensional periodic grid onto 2D:
```glsl
// 5D grid projected to 2D
float pattern = sin(uv.x * freq + uv.y * freq * phi) * 
                sin(uv.x * freq * phi + uv.y * freq);
```
The most mathematically rigorous quasiperiodic moiré.

## 4. Quasicrystal Diffraction
Simulate the diffraction pattern of a quasicrystal (the "spot" pattern seen in X-ray crystallography). The moiré becomes a *Fourier transform* of quasiperiodic order — bright spots at irrational angles.

## 5. Substitution Tiling Moiré
Use substitution rules (like the Penrose inflation rules) to generate multi-scale grids. The moiré is *self-similar* — zoom in, and you see the same interference at smaller scales.

## 6. Danzer Tiling
3D quasiperiodic tiling projected to 2D. The moiré has icosahedral symmetry traces — 12-fold and 20-fold interference zones.
